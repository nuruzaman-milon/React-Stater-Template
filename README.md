
# React Starter Template

This is a starter template for a react project. You can just clone this template, follow the command and you are good to go. 




## Running command

To run this template, run the following command

For npm: 
```bash
  npm install
  npm start
```
For yarn: 
```bash
  yarn install
  yarn start
```
